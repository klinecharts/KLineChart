# getBarSpaceLimit()
`getBarSpaceLimit` 获取图表单根蜡烛柱所占空间最大值/最小值。

## 参考 {#reference}
<!-- @include: @/@views/api/references/instance/getBarSpaceLimit.md -->

### 参数 {#parameters}
`getBarSpaceLimit` 不接收任何参数。

### 返回值 {#returns}
`getBarSpaceLimit` 返回一个包含单根蜡烛柱所占空间信息的对象 `BarSpaceLimit` 。
