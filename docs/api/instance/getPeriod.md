# getPeriod()
`getPeriod` 获取当前周期。

<!--@include: @/@views/api/references/instance/getPeriod.md-->

### 参数 {#parameters}
`getPeriod` 不接收任何参数。

### 返回值 {#returns}
`getPeriod` 返回一个包含周期信息的 `Period` 对象或者 `null` 。
