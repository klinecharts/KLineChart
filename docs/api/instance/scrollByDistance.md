# scrollByDistance(distance, animationDuration?)
`scrollByDistance` 滚动指定的距离。

## 参考 {#reference}
<!-- @include: @/@views/api/references/instance/scrollByDistance.md -->

### 参数
- `distance` 滚动距离。
- `animationDuration` 动画持续时间，如果小于等于0，则无动画。

### 返回值
`scrollByDistance` 返回 `undefined` 。
