# setBarSpace(space)
`setBarSpace` 设置图表单根蜡烛柱所占的空间大小。

## 参考 {#reference}
<!--@include: @/@views/api/references/instance/setBarSpace.md-->

### 参数
- `space` 空间大小，范围在 1 到 50 之间。

### 返回值
`setBarSpace` 返回 `undefined` 。
