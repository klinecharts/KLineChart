# Skills
KLineChart 提供了 skills 以帮助使用者在 Cursor 等支持 Agent Skills 的 IDE 中更好的使用组件。

## 文件结构
| 文件 | 说明 |
| --- | --- |
| [`SKILL.md`](https://github.com/klinecharts/KLineChart/blob/main/skills/klinecharts/SKILL.md) | 主技能：最小示例、数据加载、框架集成、指标/覆盖物、v9 迁移、排错 |
| [`api-reference.md`](https://github.com/klinecharts/KLineChart/blob/main/skills/klinecharts/api-reference.md) | v10 常用 API 速查表 |
| [`examples.md`](https://github.com/klinecharts/KLineChart/blob/main/skills/klinecharts/examples.md) | 可复制的完整代码示例 |
| [`prompts.md`](https://github.com/klinecharts/KLineChart/blob/main/skills/klinecharts/prompts.md) | 常见需求的提示词模板 |

## 如何使用
```bash
npx skills add klinecharts/KLineChart
```
