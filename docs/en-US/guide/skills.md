# Skills

KLineChart provides skills to help users work with the library more effectively in IDEs that support Agent Skills, such as Cursor.

## File structure

| File | Purpose |
| --- | --- |
| [`SKILL.md`](https://github.com/klinecharts/KLineChart/blob/main/skills/klinecharts/SKILL.md) | Main skill: minimal example, data loading, framework integration, indicators/overlays, v9 migration, troubleshooting |
| [`api-reference.md`](https://github.com/klinecharts/KLineChart/blob/main/skills/klinecharts/api-reference.md) | v10 API quick reference |
| [`examples.md`](https://github.com/klinecharts/KLineChart/blob/main/skills/klinecharts/examples.md) | Full copy-paste examples |
| [`prompts.md`](https://github.com/klinecharts/KLineChart/blob/main/skills/klinecharts/prompts.md) | Prompt templates for common tasks |

## How to use

```bash
npx skills add klinecharts/KLineChart
```
