# getBarSpace()
`getBarSpace` 获取图表单根蜡烛柱所占空间信息。

## 参考 {#reference}
<!-- @include: @/@views/api/references/instance/getBarSpace.md -->

### 参数
`getBarSpace` 不接收任何参数。

### 返回值
`getBarSpace` 返回一个包含单根蜡烛柱所占空间信息的对象 `BarSpace` 。
