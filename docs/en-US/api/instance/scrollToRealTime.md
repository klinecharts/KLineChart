# scrollToRealTime(animationDuration?)
`scrollToRealTime` 滚动最新数据的位置。

## 参考 {#reference}
<!-- @include: @/@views/api/references/instance/scrollToRealTime.md -->

### 参数
- `animationDuration` 动画持续时间，如果小于等于0，则无动画。

### 返回值
`scrollToRealTime` 返回 `undefined` 。
