# setLocale(locale)
`setLocale` set the locale for the chart.

## Reference {#reference}
<!--@include: @/@views/api/references/instance/setLocale.md-->

### Parameters
- `locale` Locale name, `zh-CN` and `en-US` are built-in.

### Returns
`setLocale` returns `undefined` .
