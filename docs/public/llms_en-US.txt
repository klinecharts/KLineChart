# KLineChart

Highly customizable, professional, lightweight financial charting library for the web, built with HTML5 canvas and TypeScript. This `llms.txt`-style file is intended for large language models (LLMs) and tools that want to answer questions or generate code for KLineChart.

## 1. Overview

- KLineChart renders high‚Äëperformance candlestick (K‚Äëline) and related financial charts in the browser with zero runtime dependencies.
- It focuses on **lightweight and smooth rendering**, **rich built‚Äëin indicators and overlays**, and **strong extensibility** via plugins and custom drawing.
- The project is written in **TypeScript**, ships type definitions, and exposes a clear **chart API** and **instance API**.
- Official docs live locally under `docs/` and online at `https://www.klinecharts.com`.

## 5. Data Model (from docs/en-US/guide/data-source.md)

Candlestick data objects should generally look like:

- `timestamp` (number, required) ‚Äì Millisecond UNIX timestamp per bar.
- `open`, `close`, `high`, `low` (number, required).
- `volume` (number, optional).
- `turnover` (number, optional, required for some indicators such as `EMV` and `AVP`).

## 6. Built‚Äëin Indicators (from docs/en-US/guide/indicator.md)

Representative built‚Äëin indicators include:

- Moving‚Äëaverage style: `MA`, `EMA`, `SMA`, `BBI`, `BOLL`
- Momentum/oscillator style: `MACD`, `KDJ`, `RSI`, `TRIX`, `AO`, `MTM`, `WR`
- Volume/flow style: `VOL`, `OBV`, `VR`, `PVT`, `AVP`, `EMV`
- Trend/structure style: `DMI`, `SAR`, `CR`, `DMA`, `ROC`, `PSY`, `BRAR`, `CCI`

Custom indicators are registered via `registerIndicator` (chart‚Äëlevel) and used via `createIndicator` (instance‚Äëlevel). See `docs/en-US/guide/indicator.md` and examples in `src/extension/indicator/`.

## 7. Representative API Examples (from docs/@views/api/samples)

These examples are adapted directly from the official samples in `docs/@views/api/samples/` and `docs/en-US/api/*`. Prefer them over inventing new patterns.

### 7.1 Initialize a basic chart and load data

Source: `docs/@views/api/samples/init-basic/index.js`

```ts
import { init } from 'klinecharts'

const chart = init('init-basic-chart')

chart.setSymbol({ ticker: 'TestSymbol' })
chart.setPeriod({ span: 1, type: 'day' })
chart.setDataLoader({
  getBars: ({ callback }) => {
    fetch('https://klinecharts.com/datas/kline.json')
      .then(res => res.json())
      .then(dataList => {
        callback(dataList)
      })
  }
})
```

### 7.3 Create indicators on the chart

Source: `docs/@views/api/samples/createIndicator-basic/index.js`, docs: `docs/en-US/api/instance/createIndicator.md`

```ts
import { init } from 'klinecharts'

const chart = init('createIndicator-basic-chart')

// Create MA on the candle pane (overlay)
chart.createIndicator('MA', { pane: { id: 'candle_pane' } })

// Create a volume indicator on a separate pane
chart.createIndicator('VOL')

chart.setSymbol({ ticker: 'TestSymbol' })
chart.setPeriod({ span: 1, type: 'day' })

  getBars: ({ callback }) => {
    fetch('https://klinecharts.com/datas/kline.json')
      .then(res => res.json())
      .then(dataList => {
        callback(dataList)
      })
  }
})
```

### 7.4 Programmatic scrolling

Source: `docs/@views/api/samples/scrollByDistance/index.js`, docs: `docs/en-US/api/instance/scrollByDistance.md`

```ts
import { init } from 'klinecharts'

const chart = init('scrollByDistance-chart')

chart.setSymbol({ ticker: 'TestSymbol' })
chart.setPeriod({ span: 1, type: 'day' })
chart.setDataLoader({
  getBars: ({ callback }) => {
    fetch('https://klinecharts.com/datas/kline.json')
      .then(res => res.json())
      .then(dataList => {
        callback(dataList)
      })
  }
})

setTimeout(() => {
  // Scroll right by 200 logical units over 200 ms
  chart.scrollByDistance(200, 200)
}, 5000)
```
